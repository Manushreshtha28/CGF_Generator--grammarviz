Using Grammarviz

// Context-Free Grammar Derivation Tool — Reference Guide

// 01

Write Grammar Rules

Enter productions in the sidebar. Format: A → BC | d. One non-terminal per line.

S → AB | a
A → aA | a
B → bB | b
// 02

Set Input String

Enter the string you want to check. Use ε for empty string productions. Characters are tokenized as you type.

// 03

Choose Derivation Mode

L-Most: Always expands the leftmost non-terminal first.

R-Most: Always expands the rightmost non-terminal.

Both: Shows both derivation types side by side.

// 04

Read the Output

Derivation Steps: Each step card shows the sentential form with the expanded symbol highlighted.

Parse Tree: Zoomable SVG, exportable to file.

Live Animation: Step-by-step playback with accept effect.
